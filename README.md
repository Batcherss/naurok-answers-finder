### Функционал

<img src="./assets/functions.jpg" width="480"/>

* **Поддержка LLM API:** Интеграция с Gemini и Groq. Приоритетная оптимизация под Groq для мгновенной генерации ответов.
* **Vision-модуль (VLM):** Передача изображений на VL-модели для распознавания текста.
* **Авто-детекция:** Расширение самостоятельно обнаруживает содержимое страницы и извлекает вопросы без ручного взаимодействия.
* **AI Debug:** Отображает затраченное время на генерацию и количество использованных токенов в реальном времени.
* **Автовход:** Автоматически входит в тест — без ручного запуска.
* **Инъекция кода:** Глубокое внедрение в JS и CSS среду целевого сайта.
* **Time-Bypass:** Алгоритм сокращения времени на анимации «Успех / Ошибка».
* **Точность:**
    * Текст и математика — до **90%**
    * Графика — до **60%** *(ограничения бесплатных нейросетей)*
